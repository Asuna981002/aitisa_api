Applies a pooling over an input signal composed of several feature maps.

Parameters
inputThe input tensor of a shape [num_batches, in_channels, in_spatial_shapes].
modeThe pooling mode including "max" and "avg".
ksizeThe window size of a shape each of which is >= 1.
strideThe stride of a shape [spatial_shapes] each of which is >= 1.
paddingThe padding of a shape [spatial_shapes] each of which is >= 0. If the padding number of some dimension is non-zero, implicit zero padding is added to input on both sides.
outputThe output tensor pointer of a shape [num_batches, out_channels, * * out_spatial_shapes].
Returns
Status The Status enum indicates whether the routine is OK.
